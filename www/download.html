Getting GEF

CVS

This is the latest development distribution of GEF. The code in this distribution compiles independently of ArgoUML, but is not currently feature complete. To get GEF from CVS, set your CVSROOT environment variable like this:

CVSROOT=:pserver:guest@cvs.tigris.org:/cvs

Then log in to cvs as guest (password is guest):

cvs login

And check out the GEF distribution:

cvs checkout gef/dist

This distribution uses ANT (http://jakarta.apache.org) for its build system. ANT is a cross-platform build tool based on Java and XML, similar to 'make'. Read the INSTALL.txt file in the distribution.for details on how to build GEF with ANT.

 

Other distributions:

ArgoUML v.070

The latest stable source code for GEF is embedded in the ArgoUML distribution. You can download ArgoUML here.

 

GEF v.060

This was the last independent distribution of GEF. It is significantly out of date compared with the latest ArgoUML v.070 code. If you're still interested in it, you can download it from the old GEF web site.
