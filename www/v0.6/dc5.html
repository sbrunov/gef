Final Step: Get Started with GEF
 


To Run GEF:

  • Unzip gef060.zip to get gef060.jar, and unzip XML4J.zip to get xml4j_1_1_14.jar. Do not unarchive these *.jar files.
  • If you downloaded swing103.zip then you will have to unzip it to get swing-1.0.3/swing.jar
  • Start a new command line window.
  • Set your CLASSPATH to include the following in the given order:
    • classes.zip from your JDK distribution
    • gef060.jar
    • swing-1.0.3/swing.jar
    • xml4j_1_1_14.jar
  • Type the command:
         java uci.gef.demo.BasicApplication
    or on some Windows Java implementations type:
         java -classpath %CLASSPATH% uci.gef.demo.BasicApplication


Keep in Mind:

GEF is a research prototype. It will evolve into a full featured graph editing tool in the future, but it is not comparable to commercial libraries yet. If you believe in the importance of an open-source components, please help in the development of GEF.
