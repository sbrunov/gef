Getting GEF

CVS

This is the latest development distribution of GEF. The code in this distribution compiles independently of ArgoUML, but is not currently feature complete. To get GEF from CVS, set your CVSROOT environment variable like this:

CVSROOT=:pserver:username@cvs.tigris.org:/cvs

If you have been given write access to the CVS repository, then replace 'username' with your own login name. Otherwise, use the username 'guest' (password is 'guest') to get read access to the CVS repository. Then log in to CVS with your password:

cvs login

And check out the GEF distribution:

cvs checkout gef/dist

This distribution uses Ant (http://jakarta.apache.org/ant) for its build system. Ant is a cross-platform build tool based on Java and XML, similar to 'make'. Read the INSTALL.txt file in the distribution.for details on how to build GEF with Ant.

You can also browse CVS source code over the web.

For more information on using CVS, documentation, and CVS clients, click on the "Tool Docs" menu at left and select the "Source/CVS" option. Or, click here.
