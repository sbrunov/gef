Join the GEF Project

Joining the GEF project is easy: just send email to jrobbins@tigris.org to discuss the tasks you are interested in doing.

Once we work out a task, it will be added to the Tasks page or your name will be added to an existing task. When you complete the task it will be added to the next release of GEF.

The way most people contribute changes is to email a "patch" to one of the "module owners". These are developers who are responsible for certain sections of the code (called modules). There is a list of module owners here. Once a module owner sees your contribibution, he or she will review it and integrate it into the source code repository.
