Tasks

Here are some ideas for enhancements to GEF. In building an application that uses GEF you will definately feel the desire to add new features. I would like to accumulate the various enhancements developed by different people so that we can all avoid reinvention. Also, university students looking for class projects might consider the following ideas.If you plan on implementing any of these ideas, please contact me so I can tell you if anyone else is working on it already.
 
Ideas for Infrastructure Improvements
  • Support for undo
  • Support for cut and paste
  • Allow applets to be wrapped as DiagramElements
  • Load and save diagrams in multiple formats
  • Formalize/implement the concept of a Document
  • Port this framework to Java 1.1 and Bravo
  • Build in debugging support for people writing Actions, Modes, etc.
  • High-level state transition to specify all transitions among Modes
  • DrawContext object that wraps java.awt.graphics, supports scrolling, scaling, dimming
  • Drag and Drop between windows
  • Add concept of variable drawing quality: first draw fast and cheap, then come back and redraw to look better
Ideas for New Features
  • Better arc routing
  • New ArcPerspectives: direct arcs, rectilinear arcs, curved arcs
  • "Clean up" action that automatically lays out part of a diagram
  • Read-only mode (easy)
  • User visible commands for dealing with multiple layers
  • Fill and line paterns
  • Replicate command to make many copies of DiagramElement (easy)
  • Grid dialog box to set grid size, colors, origin, etc. (easy)
  • Menu items to lock layers, gray out layers
  • User visible support for macros via ActionSequence
  • Add javadoc comments that have APPLET tags to demostrate the features being commented
  • User level documention for all existing Actions, Modes, etc.
Ideas for Applications
  • OMT editor. Ideally the OMT model could be parsed from actual java source code or the output of javap.
  • Process modeling tool
  • Web site mapper (exciting!)
  • College course prerequiste editor that indicates degree progress
  • Corporate organization chart with links to email or home pages
  • Office floorplan editor with arcs for cables and power lines
  • RDBMS schema editor
  • Pinball construction tool (fun!)
