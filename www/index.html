   1.1.7 with Swing 1.0.3		   GEF is a free Java library for use in the development of new connected graph user interfaces. GEF can be used in any Java program that requires the ability to display and edit connected graphs or structured graphics.

GEF provides substantial functionality and a high quality user experience, yet is easy to learn and to extend.
