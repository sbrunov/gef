Step Two: System Requirementsnts
 


ATTENTION !!

The code you are going to download is declared unstable. It may burn your CPU and melt your harddisk. If you want to use GEF as base to your every day work, use a stable release instead.

You will need to have the following before you can use GEF:

  • A Java 1.2+ Virtual Machine
  • A JAXP compliant XML-Parser (use of Xerces is recommended)
    • Xerces-J-bin.1.2.0.zip (you can get source for this library from the Apache XML Project)
